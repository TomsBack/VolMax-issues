# VolMax — Volume Booster & EQ

Bug reports and feature requests for the **VolMax** browser extension.

## What is VolMax?

VolMax boosts any tab's volume up to 1000% with a built-in 6-band equalizer, per-tab memory, profiles, and keyboard shortcuts.

**Features:**
- Volume boost up to 1000%
- 6-band equalizer with presets (Flat, Bass, Vocal, Treble, V-Shape)
- Left/right balance control
- Night mode, normalize, mono, cut rumble, low-pass filter
- Per-tab settings memory
- Saveable profiles with auto-save
- Keyboard shortcuts (Alt+Shift+Arrow / M)
- Badge & favicon indicators

## Report a bug

[Open a new issue](https://github.com/TomsBack/VolMax-issues/issues/new) and include:
1. Browser name and version
2. What you expected to happen
3. What actually happened
4. Steps to reproduce

## Request a feature

[Open a new issue](https://github.com/TomsBack/VolMax-issues/issues/new) with the **feature request** label.

## Privacy

VolMax does not collect any data. No analytics, no tracking, no network requests. Everything is processed locally. [Read the full privacy policy](https://tomsback.github.io/VolMax-issues/privacy-policy.html).
